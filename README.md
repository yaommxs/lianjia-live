# lianjia-live
看链家——成都地区爬虫数据实时直播平台
#### 技术栈
前端：react+antd+echarts  
后台：node.js+express+mysql  
API：axios  
#### 在线地址：[http://lianjia.yaommxs.com]('http://lianjia.yaommxs.com')  
#### 功能：  
+ 各区域菜单选择
+ 各图表可视化展示
+ 对应文案说明
+ 房源信息按类别排序功能
+ 房源信息模糊搜索功能
+ 数据分页展示
+ 各链接跳转功能
+ 全屏展示
+ 回到顶部展示

#### 注：
数据每日更新,时间可能会有延迟，信息仅供参考，链接说挂就挂，感谢阅读！
